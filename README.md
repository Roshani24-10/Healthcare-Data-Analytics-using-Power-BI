# Healthcare-Data-Analytics-using-Power-BI
Power BI dashboard analyzing healthcare patient data, hospital performance, billing insights, and insurance trends.
## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization

## Key Insights
- Patient distribution by gender and age group
- Hospital performance based on billing
- Insurance provider billing comparison
- Medical condition analysis
- Admission type trends

## Files Included
- Healthcare_Dashboard.pbix (Power BI dashboard)
- healthcare_dataset.csv (dataset used for analysis)

## Dashboard Features
- KPI cards for key metrics
- Interactive slicers
- Multiple visualizations including bar charts, donut charts, and line charts
